📌 数据计算说明（基于您提供的CSV）
1. 最新一周：取 fact_order 中最大 order_time 所在周（2025-03-10）。
2. KPI 计算：GMV = sum(pay_amount)，ordering users = nunique(user_id)，orders = count(order_id)，AOV = GMV/orders，gross margin = (GMV - sum(cogs))/GMV，contribution profit = gross profit - fulfillment_cost - allocated_marketing。
3. 渠道结构：按 week 和 acquisition_channel 聚合 ordering users，得到堆叠面积图数据。
4. 品类结构：从 bridge_order_item 与 dim_item 关联，按 category 聚合 GMV，得到饼图数据。
5. 漏斗：注册用户 = dim_user 总数，首购用户 = fact_order 中每个 user_id 首次订单的去重数，30天复购用户 = 首购后30天内再次下单的用户数。
6. 渠道效率：CAC = channel_cost / channel_users，LTV = 用户首购后90天内 contribution_profit 之和的平均值。
您可以在 Python 环境中运行提供的 ipynb 代码，将计算出的数组替换上面 JavaScript 中的对应部分。
